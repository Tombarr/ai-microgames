- Puzzle game enthusiasts

- Fans of minimalist design
- Pattern recognition aficionados

### Gameplay Loop

1. 4x4 grid appears with pipes in wrong positions
2. Player clicks/taps to rotate segments 90¬∞ clockwise
3. Goal: All pipes must connect with no dead ends
4. 5-second timer creates urgency
5. Win = advance with faster speed multiplier
6. Lose = restart (limited lives in campaign mode)

### Technical Details

- **Engine:** Godot 4.5.1
- **Platform:** Web (HTML5), exportable to desktop/mobile
- **Resolution:** 640x640 pixels
- **File Size:** < 2MB
- **Input:** Mouse, Touch, Keyboard
- **Accessibility:** Colorblind-friendly (no color-coded gameplay)

- Pipe Dream / Pipe Mania (rotation mechanics)
- WarioWare series (microgame format)
