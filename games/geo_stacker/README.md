# GeoStacker

A fast-paced 2D block stacking game where you must stack vibrant, geometric shapes to reach a target
height before time runs out.

## How to Play

- **Objective:** Stack the shapes to reach the target line before the timer runs out.
- **Controls:**
  - **Move:** Use the left and right arrow keys to move the shape.
  - **Rotate:** Use the up arrow key to rotate the shape.
  - **Drop:** Press the spacebar to drop the shape.
- **Scoring:** You win the game by reaching the target line. You lose if the timer runs out or if a
  shape falls off the stack.
